Update on Glasses Mount

Please note that the circular glasses mount shown in the pictures has been updated. I designed a Version 2 with a more rectangular shape to prevent the glasses from moving sideways. The STL file included in the download is this improved version.

Project Overview

I made three wall mounts to organize my headphones, projector remote, and glasses next to my bed. The goal was to keep everything within reach while keeping the area tidy.

The mounts can be attached using double-sided tape (recommended) or with a screw using the built-in mounting hole for a more secure fit.

I’m using Sony WH-1000XM3 headphones, although the headphone mount is universal and works with any other headphones. The remote mount is designed for this projector remote: https://www.amazon.com/dp/B0DX5HR76H

The remote mount can be scaled to fit other remote sizes if needed.
