I made these simple brackets to adjust the height of an existing shelf in my Ikea Detolf display case. This design allows for adjusting the shelf height, which is perfect if you want to keep large models inside. The pieces were designed in Autodesk Tinkercad.

Inspiration & Use Case

I made this design specifically to elevate one of the shelves so that my Lego Avengers Tower would fit properly in the shelf. I came up with this bracket design as a solution to adjust the shelves to fit larger models.

Installation

To install these brackets, you will need:

  • Flat head machine screws and nuts
  • Double-sided tape for extra support (highly recommended on top of the bracket)
