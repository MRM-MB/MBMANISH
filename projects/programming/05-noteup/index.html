NoteUp

  • Python
  • Javascript
21 Aug 2024
🔗 Live Demo GitHub

🚀 Experience the app in action or explore the source code

NoteUp started as a personal fix for handwritten account notes that were hard to search. I wanted a local app where people can store information, search quickly, and view accounts in a clean, visual layout.

Highlights

  • Accounts Dashboard: Add, update, and delete saved accounts.
  • Secure Access: Registration, login, and password recovery.
  • Convenience: Google OAuth and automatic inactivity logout.
  • Company Branding: Personalized banners and icons per company, with a smart fallback if assets are missing.

If a company logo or banner isn’t available, NoteUp shows a default asset and prompts you to check the company spelling. You can add your own assets using the README steps. For high‑quality brand resources, I use Brandfetch.

Ways to try it

  • Live Demo: Hosted on Render for a quick preview. Data resets on restarts.
  • Run Locally: Recommended for persistent storage and full privacy.

For setup steps, environment details, folder structure, and technical notes, please refer to the project's GitHub README.
