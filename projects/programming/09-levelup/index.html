LevelUp is a 3rd-semester project developed at the University of Southern Denmark (SDU) in collaboration with the Danish company LINAK.

Project Overview

The goal was to build a full-stack system that helps office users follow the Cornell 20:10 sit-stand cycle, track activity, and manage adjustable desks through a web application with gamified rewards and clear usage insights.

The solution combines a Laravel web app, a MySQL database, and a desk simulator to coordinate desk control and user activity.

Tech Stack

Built with PHP, JavaScript, Blade, CSS, MicroPython, plus Dockerfile and Shell for setup scripts. The Pico W firmware uses MicroPython, flashed via Thonny.

Hosted Demo (Desk Simulator + App)

Open the simulator first, keep it running, then open the app (it reads from the hosted simulator).

Open the simulator · Open the LevelUp app

Demo limits: No Pico W hardware, demo data resets (new accounts are temporary), and you cannot use the hosted demo with a Pico W. For Pico W use, run the app locally and set WIFI2BLE_BASE_URL (see SETUP.md in the README).

Hardware Integration (Raspberry Pi Pico W)

The system includes a Raspberry Pi Pico W controller for real-time desk feedback, including OLED status, RGB phase indicators, and a physical pause button.
